#  Grocery Store Management System (SQL Project)

##  Project Overview
This project is a **Grocery Store Management System** developed using **SQL** to manage and organize grocery store operations efficiently.  
It focuses on handling **products, customers, orders, inventory, and billing data** using structured database design and SQL queries.

The project is mainly created for **learning, academic purposes, and interview preparation**.

---

##  Objectives
- Design a well-structured relational database
- Perform CRUD (Create, Read, Update, Delete) operations
- Manage inventory and sales data efficiently
- Practice SQL queries for real-world business scenarios
- Improve database and query-writing skills

---

##  Database Features
- Product management
- Customer information storage
- Order and billing records
- Inventory tracking
- Sales analysis using SQL queries
